## 如何回测

```bash
cd backtest && python3 run_backtest.py
```

**回测设计要点(防过拟合三件套)**：
1. **In-sample(样本内) / Out-of-sample(样本外) 切分**：以研报发布日 2011-07-11 为界。样本内是研报作者"看过"的数据，样本外才是真实检验。**所有参数(MA窗口=4、换手率滞后=4、PCA窗口=156周、阈值±2σ)只允许在样本内调整**，样本外只跑一次、不回头改参数；
2. **无前视偏差(look-ahead bias)**：t 周五收盘算信号 → t+1 周开盘成交；滚动 PCA 与滚动阈值均只用截至当期的数据；
3. **成本**：单边 6bp(ETF 口径)。改 `COST_ONEWAY` 可做成本敏感性。

**当前回测结论(诚实汇报)**：用价量代理变量复现后，策略样本外年化 1.4%，跑输沪深300 buy-and-hold(2.8%)，但波动更低。说明 GMX 的择时增益主要来自原文的**场外资金类变量(开户数/IPO)**，纯价量代理不足以复制——这是后续优化的明确方向，不是 bug。

## 代理变量映射(与原文的差异)

| 原文变量 | 数据源 | 本复现代理 | 理由 |
|---|---|---|---|
| PE / PB | Wind | 价格/52周均线偏离 | 估值水平的价格近似 |
| 换手率(-4) | Wind | 周成交额/52周均额，滞后4周 | 同为活跃度，TDX有成交额 |
| 上涨家数占比 | Wind | 近4周上涨周数占比 | 市场宽度近似 |
| IPO 首日涨幅 | Wind | 动量×波动 | 投机热度近似 |
| 新增开户数 | 中登公司 | 26周动量 | 场外资金为趋势确认变量 |
| (小盘相对收益) | Wind | 中证500/沪深300 相对强弱 | 直接可得 |

## 如何实盘

1. **频率**：周频。每周五收盘后运行：

```python
import sys; sys.path += ["../common", "."]
import tdx_data
from sentiment_index_strategy import run_pipeline
w300 = tdx_data.to_weekly(tdx_data.index_daily("000300", 1, refresh=True))
w500 = tdx_data.to_weekly(tdx_data.index_daily("000905", 1, refresh=True))
sig = run_pipeline(w300, w500)
print(sig.tail(1))   # position=1 -> 下周一开盘买入/持有沪深300ETF(510300)；=0 -> 空仓
```

2. **标的**：沪深300ETF(510300/159919)，或 IF 股指期货(注意保证金与基差)；
3. **执行**：信号出在周五收盘后，**下周一开盘**执行，与回测口径一致；
4. **风控**：该策略无止损逻辑，靠 GMX 上界做风险提示；建议叠加账户级最大回撤限制(如 -15% 强制减半仓)；
5. **监控**：GMX 长期贴着上界/下界跑说明阈值失效(原文 2014 年牛市就发生过类似情形)，需检查代理变量分布是否漂移。

## 风险提示

- 情绪变量结构性失效风险(原文的开户数、封基折价率均已失效)；
- 阈值规则历史触发次数少，统计显著性弱；
- 滚动 PCA 起算需要 156 周 + 52 周特征预热，新市场/新变量上线后约 4 年内无信号。
