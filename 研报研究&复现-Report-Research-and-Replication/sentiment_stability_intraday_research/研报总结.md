## 核心思想

交易时间只有 270 分钟，非交易时间 1170 分钟——**隔夜消息密度远高于盘中**，因此当日趋势大概率在早盘已定型。用早盘走势判断"当日市场情绪受消息面影响的方向与强度"，趋势日做 *trend following(趋势跟随)*，震荡日不交易。关键问题：**如何区分趋势日与震荡日**。

这是广发"另类交易策略"系列的第三代方法：NTT(混沌理论+LM非线性拟合) → EMDT(经验模态分解信噪比) → 本篇**平稳度指标**。前两代有效但数学繁琐、缺乏金融直觉；本篇用"回撤"思想一行公式解决。

## 平稳度指标定义

对早盘价格序列 `{p1,...,pn}`：
- 每点 **回撤** `DD_i = max_{j≤i}(p_j − p_i)/p_j`(从前高回落幅度)；
- 每点 **反向回撤** `RDD_i = −min_{j≤i}(p_j − p_i)/p_j`(从前低回升幅度)；
- **平稳度 = min(平均回撤, 平均反向回撤)**。

直觉：单边上涨时回撤均值≈0(平稳度小)；单边下跌时反向回撤均值≈0(平稳度也小)；震荡时两者都大。**平稳度越小，趋势性越强**。例：趋势日 min(0.002234, 0.000252)=0.000252 → 开仓；震荡日 min(0.0018, 0.0018)=0.0018 → 放弃。

## 基础策略(单次开仓)

1. 观察开盘后 **50 分钟**(样本内优化：窗口太长开仓晚吃不到收益，太短判断不准)；
2. 平稳度 < **9/10000** → 当日可交易；否则空仓；
3. 方向：50 分钟时价格 > 开盘价 → 做多，反之做空；
4. 离场：**止损 0.5%**，否则持有到收盘平仓。

## 改进策略(日内两次开仓)

动机：2014 年起日内宽幅震荡加剧，**上午与下午行情相关性减弱**(上午趋势下午跳水 / 早盘震荡尾盘拉升)，单次开仓模型失效。

改进：上午、下午各独立决策一次——
- **上午**：同基础策略，但 **11:30 午盘收盘即平仓**；
- **下午**：用 11:12~13:32 样本(跨午休，含上午尾盘信息)计算平稳度，**13:33 决定是否开仓**，方向同理，收盘平仓。

开仓频率的权衡：样本要够长(判断可靠)、持仓要够长(覆盖成本)、参数要少(防过拟合)——所以只拆两段，不做更高频。

## 实证结果

数据：当月 IF 股指期货 1 分钟收盘价；成本双边 2/10000；止损 0.5%。
**样本内** 2010-04-16 ~ 2012-12-31(参数优化)；**样本外** 2013-01-04 ~ 2015-02-25(只跑一次)。

| 指标 | 单次开仓(IS/OOS) | 两次开仓(IS/OOS) |
|---|---|---|
| 累计收益 | 137.3% / 28.1% | 169.0% / **31.6%** |
| 年化收益 | 38.7% / 12.8% | 45.4% / **14.2%** |
| 最大回撤 | −5.7% / −5.4% | −4.7% / **−4.7%** |
| 胜率 | 46.9% / 49.5% | 51.8% / 49.5% |
| 赔率 | 2.29 / 1.60 | 1.59 / 1.36 |
| 交易次数 | 324 / 196 | 720 / 465 |

对开仓阈值、开仓时间均做了**参数平原(parameter plateau)测试**，表现稳定——这是研报质量的重要加分项。

## 评价与局限

**价值**：平稳度指标是**极简且有金融直觉**的趋势/震荡过滤器(对比 NTT/EMD 的复杂数学)；研究流程规范——明确 IS/OOS 切分、含成本与止损、参数敏感性完整；"消息密度不对称"的开仓逻辑至今成立。

**局限**：
1. 2015-09 股指期货受限后(开仓手数、保证金、平今仓手续费 23bp)，**双边 2/10000 的成本假设彻底失效**，策略原样照搬必亏；2023 年后虽有松绑，平今成本仍高；
2. 典型的牛市后段样本外(2013-2015)，未经历 2016 熔断、2018 单边熊检验；
3. 胜率≈50%，盈利全靠赔率(截断亏损)，对滑点敏感；
4. 趋势日占比是策略产能上限，市场波动率长期下行会压缩交易机会。

**复现注意**：通达信免费服务器 1 分钟线仅保留最近约 4-5 个月，复现用**沪深300指数 1 分钟线**替代 IF 当月连续(无期货主力合约拼接数据)，结果只能验证逻辑通路与近期表现，不能复刻原文 2010-2015 区间。
