## 如何回测(in-sample / out-of-sample)

- **切分**：样本内 2010-06 ~ 2020-02-24(研报发布日)；样本外 2020-02-24 ~ 至今。**原文没有做 IS/OOS 切分(其最大缺陷)，复现把这一课补上了**；
- **当前回测结论**：
  - 相对收益版样本内年化超额 **8.8%**(原文 11%，代理变量下量级吻合)，信息比 0.52；
  - **样本外(2020-02 至今)依然有效**：策略超额 +1.1%/年 vs 持有超额 −3.3%/年，相对回撤从 −32.6% 压缩到 −13.0%——该策略的价值核心是**躲过券商板块的漫长阴跌**，与原文"负面事件必须显著"的设计哲学一致；
  - 绝对收益版全区间年化 8.3% vs 券商指数躺平 0.8%，最大回撤减半；
- **口径**：月频，t 月末信号 → 次月首日开盘成交，单边 15bp；
- 敏感性测试：改 `broker_cycle_strategy.py` 的 `OPEN_K / CLOSE_K / PB_WINDOW`，注意**只能在样本内调**，调完后样本外只许跑一次。

## 如何实盘

1. **频率**：月频。每月最后一个交易日收盘后：

```python
import sys; sys.path += ["../common", "."]
import tdx_data
from broker_cycle_strategy import build_indicator, generate_signals
monthly = lambda d: d.resample("ME").agg({"open":"first","close":"last","amount":"sum"}).dropna()
broker = monthly(tdx_data.index_daily("399975", 0, refresh=True))
sh = monthly(tdx_data.index_daily("999999", 1, refresh=True))
sz = monthly(tdx_data.index_daily("399001", 0, refresh=True))
ind = build_indicator((sh["amount"]+sz["amount"]).reindex(broker.index), broker["close"])
print(generate_signals(ind["indicator"]).tail(1))  # position=1 -> 次月初买入券商ETF
```

2. **标的**：券商 ETF(512880 / 512000)，流动性充足；相对收益版需同时空 IF 或减配沪深300仓位；
3. **执行**：信号在月末，次月**第一个交易日开盘**执行，与回测一致；
4. **升级方向**(按预期收益排序)：
   - 接入真实月度两融余额、IPO/增发规模(akshare 可得)，还原原文三项业务收入估算；
   - 用申万证券指数真实 PB(理杏仁/akshare)替换价格代理；
   - 原文事件化框架可平移到保险(盈利驱动)、白酒等其他行业，构建行业景气度组合。

## 风险提示

- 年均仅约 2 次开仓，单次信号权重高，错一次影响全年；
- 佣金率下行 + 财富管理转型使"成交额→券商收入"弹性逐年衰减，代理有效性需每年校验(滚动相关性监控)；
- 行业政策冲击(如 2023 降费、并购重组主题)会让基本面框架短期失灵——原文风险提示同样强调"行业基础逻辑变化时模型失效"。
