## 如何回测(in-sample / out-of-sample)

- **切分**：样本内 2004 ~ 2015-01-19(研报发布日)，样本外 2015-01-19 ~ 至今；
- **纪律**：窗口 242 日、1.5σ、20 日平滑全部沿用原文，样本外零调参。原文作者本就**拒绝参数优化**(情感工具有误差 + 信号样本量小没有统计显著性)，复现保持同样纪律；
- **执行口径**：t 日收盘出信号 → t+1 日开盘成交，收益从 t+1 开盘记到 t+2 开盘；单边成本 10bp；
- **多空版本**：原文假设可做空上证指数；实盘只能用 IF/IH 空头替代，回测中多空版仅作机制展示。

**当前回测结论(诚实汇报)**：
- 买入(恐惧)信号后 22 日指数平均 **+5.14%**(15 次)——逆向买入逻辑用价量代理依然成立；
- 卖出(贪婪)信号后 22 日约 0%——卖出端无效，与原文"卖出信号弱于买入信号"的不对称一致，价量代理放大了该缺陷(放量上涨在 A 股牛市中段是顺势信号而非反转信号)；
- 连续持仓的净值表现差(信号锁死问题，原文 2011、2014 年也栽在这里)；**该策略的正确用法是把信号当"底部确认器"，而不是连续仓位**；
- 行业轮动组1(最悲观)年化 10.4% > 组5(最乐观) 9.3%，方向对但价差远小于原文 10%——文本情绪含有价量之外的增量信息，这正是本研报的核心价值。

## 如何实盘

1. **频率**：日频检查，月频换行业。每日收盘后：

```python
import sys; sys.path += ["../common", "."]
import tdx_data
from fear_greed_strategy import build_sentiment_proxy, generate_signals
df = tdx_data.index_daily("999999", 1, refresh=True).loc["2004-01-01":]
sig = generate_signals(build_sentiment_proxy(df))
print(sig.tail(1))  # signal=+1 当日触发买入；-1 触发卖出；position 为当前状态
```

2. **建议用法**(基于回测证据)：
   - **只用买入信号**：触发后分批买入宽基 ETF(510300/510050)，持有约 22 个交易日或情绪回到均值即止盈；
   - 卖出信号仅作风险提示(降杠杆)，不直接做空；
3. **行业轮动**：每月第一个交易日，按上月行业情绪买入最悲观组(等权 5-6 个行业 ETF)，持有一个月；
4. **升级路径**：接入真实文本情绪(东方财富股吧/雪球 API + 情感分析)替换 `build_sentiment_proxy`，函数签名不变即可无缝替换——这是本策略最值得做的一笔投入。

## 风险提示

- **信号锁死**：触发买入后若情绪长期不触上界，会被动持有穿越下跌(原文 2011-05~11 案例)；建议加时间止损(如 44 日)或 -8% 硬止损；
- **情绪-指数背离**：结构性行情(少数行业拉指数)时大众情绪与指数脱钩(原文 2014-09 失效案例)，需监控两者滚动相关性；
- 极端阈值事件低频(年均约 1.5 次买入)，单次仓位不宜过重。
